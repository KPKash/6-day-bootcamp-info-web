Programme Curriculum

Session 01
What is Programming?
  • How programs work — instructions computers follow
  • Why Python is perfect for beginners
  • Installing Python and setting up VS Code / IDLE
  • Your first script: print("Hello, World!")
  • Understanding syntax errors and how to fix them
Session 02
Variables & Data Types
  • What is a variable? Storing information in memory
  • Data types: integers, floats, strings, booleans
  • Taking input with input() and displaying with print()
  • Arithmetic operators: +, −, *, /, // , %
  • String operations: concatenation, f-strings, len()
Session 03 · Project
Build: Simple Calculator & Greeter
  • Calculator that adds, subtracts, multiplies, divides
  • Greeting program that personalises a welcome message
  • Common beginner errors walkthrough
  • Q&A and reflection: what can code automate?
✅
Day Outcome

You'll have a working Python environment and understand how to write programs using variables, data types, operators, and user input — the same building blocks used in every programming language.

Session 01
Making Decisions in Code
  • Conditional logic: if, elif, else
  • Comparison operators: ==, !=, >, <, >=, <=
  • Logical operators: and, or, not
  • Nested conditions with real-life examples
  • Even/odd checker and grade calculator
Session 02
Loops & Repetition
  • for loops — iterating over ranges and lists
  • while loops — repeat until a condition is false
  • break and continue to control flow
  • Lists: creating, accessing, appending, looping
  • Summing a shopping list with a loop
Session 03 · Project
Build: Number Guessing Game
  • Random number generation with random module
  • Loop until the player guesses correctly
  • Count attempts and show a personalised result
  • Discussion: how do these ideas appear in real apps?
✅
Day Outcome

You can write programs that make decisions and repeat tasks automatically. These two concepts — conditions and loops — are the backbone of every piece of software ever written.

Session 01
How the Web Works & HTML Basics
  • Browsers, servers & what HTML actually is
  • Document skeleton: <html>, <head>, <body>
  • Text tags: <h1>–<h6>, <p>, <strong>, <em>, <br>
  • Opening vs self-closing tags and attributes
Session 02
Essential HTML Tags Deep Dive
  • Links: <a href="..."> — internal and external
  • Images: <img src="..." alt="..."> — paths and alt text
  • Lists: ordered <ol>, unordered <ul>, items <li>
  • Tables: <table>, <tr>, <th>, <td> with colspan/rowspan
  • Grouping tags: <div>, <span>, <section>, <nav>, <footer>
Session 03 · Project
Build: Student Bio Page
  • A profile page with heading, photo, and about paragraph
  • A linked navigation bar using <ul> and <a>
  • A table of hobbies/skills with headers and rows
  • Minimal inline CSS — just color and font-size to taste
✅
Day Outcome

You'll confidently use the core HTML tags that make up every webpage — text, links, images, lists, and tables — and understand how browsers read and render them.

Session 01
How CSS Works
  • What is CSS? The paint & styling layer of HTML
  • Ways to add CSS: inline, <style> tag, external file
  • CSS selectors: tag, class .name, ID #name
  • The cascade: which rule wins and why
  • Using browser DevTools to inspect and tweak live styles
Session 02
Essential CSS Properties
  • Colour: color, background-color, hex values and named colours
  • Typography: font-family, font-size, font-weight, text-align, line-height
  • Spacing: margin, padding — the box model explained visually
  • Borders: border, border-radius for rounded corners
  • Sizing: width, height, max-width in px and %
Session 03 · Project
Style the Bio Page End-to-End
  • Link a Google Font and apply it site-wide
  • Style headings, paragraphs, links, and list items with CSS classes
  • Add coloured backgrounds, borders, and shadows to cards
  • Style the table with alternating row colours using class selectors
  • Peer gallery: share screens and compare different styling choices
✅
Day Outcome

You'll know how to make any HTML element look polished — controlling colours, fonts, spacing, borders, and sizing with CSS properties to turn plain markup into a styled, attractive page.

Session 01
Design Principles & Visual Thinking
  • What is UX? Design that serves people, not just looks pretty
  • The five principles: contrast, alignment, proximity, repetition, balance
  • Colour theory: warm vs cool, complementary palettes, tool — Coolors.co
  • Typography basics: serif vs sans-serif, font pairing do's and don'ts
  • Analysing real designs: what works and why
Session 02
Getting Started with Photopea
  • Photopea overview — free, browser-based, Photoshop-compatible
  • Canvas setup: size presets for social media and print
  • Layers panel: creating, naming, grouping, reordering
  • Shape tool, text tool, and image placement basics
  • Exporting as PNG, JPG, and PDF
Session 03 · Design Challenge
Build: Social Media Banner
  • Create an event/announcement post for a fictional brand
  • Apply a colour palette, chosen font pair, and layout grid
  • Add icons, shapes, and a background — all in Photopea
  • Group showcase and feedback round
  • Career wrap-up: Developer, Designer, QA, Data — what's next?
✅
Day Outcome

You'll leave with a polished graphic you created, a vocabulary for talking about design, and a clearer picture of where your interests in IT actually lie — ready to decide your path.

What is AI & Machine Learning?
  • AI vs ML vs Deep Learning — the difference explained simply
  • How machines "learn": patterns, data, and predictions
  • Types of ML: Supervised, Unsupervised, Reinforcement (with everyday examples)
  • Real-world AI you already use: recommendations, voice assistants, spam filters
  • What data actually is — and why it's the fuel of AI
Session 02
Hands-On: AI Tools Without Code
  • Exploring Google Teachable Machine — train an image classifier in the browser, no code

  • Discussion: what makes an AI model good or bad?
Session 03 · Wrap-Up
AI in Careers & the Road Ahead
  • Jobs in AI/ML: Data Analyst, ML Engineer, AI Researcher, Prompt Engineer
  • What skills are needed — Python, maths, curiosity

  • What to learn next: Python for Data Science, intro to pandas & scikit-learn
  • Q&A and 5-day reflection: what excited you most this week?

You'll understand what AI and ML actually are (not just buzzwords), have trained your own image-recognition model without writing a single line of code, and know exactly what steps to take if you want to go deeper into this field.
